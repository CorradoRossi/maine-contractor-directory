## How to submit a new contractor

* Fork the repo
* Add a new contractor to `src/data/index.ts`
* Create a PR

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
